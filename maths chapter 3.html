Review of Relations and Functions
Cartesian Products, Relations, Functions and Key Definitions
  • Ordered pair → (first coordinate, second coordinate)
    • Examples: $(2,3)$, $(-2,4)$, $(a,b)$
    • First coordinates: $2$, $-2$, $a$
    • Second coordinates: $3$, $4$, $b$

Cartesian Product

  • Given sets $A = \{3, 4\}$ and $B = \{2, 4, 5\}$
    • $A \times B = \{(3,2), (3,4), (3,5), (4,2), (4,4), (4,5)\}$
Definition: Suppose $A$ and $B$ are sets. The Cartesian product of $A$ and $B$, denoted by $A \times B$, is the set which contains every ordered pair whose first coordinate is an element of $A$ and second coordinate is an element of $B$, i.e. $$A \times B = \{(a, b) : a \in A \text{ and } b \in B\}$$
  • Example: For $A = \{2, 4\}$ and $B = \{-1, 3\}$
    • $A \times B = \{(2, -1), (2, 3), (4, -1), (4, 3)\}$
    • $B \times A = \{(-1, 2), (-1, 4), (3, 2), (3, 4)\}$
    • $A \times B$ and $B \times A$ are not equal (order matters)
  • Two sets are equal if one is a subset of the other and vice versa
  • To check equality of Cartesian products we need to define equality of ordered pairs
Equality of Ordered Pairs: Two ordered pairs $(a, b)$ and $(c, d)$ are equal if and only if $a = c$ and $b = d$.
  • Example: Let $A = \{1, 2, 3\}$ and $B = \{a, b, c\}$
    • $A \times B = \{(1, a), (1, b), (1, c), (2, a), (2, b), (2, c), (3, a), (3, b), (3, c)\}$

Relations

Definition: If $A$ and $B$ are sets, any subset of $A \times B$ is called a relation from $A$ into $B$.
  • Suppose $R$ is a relation from a set $A$ to a set $B$
    • $R \subseteq A \times B$
    • For each $(a, b) \in A \times B$, either $(a, b) \in R$ or $(a, b) \notin R$
    • $(a, b) \in R$ → $a$ is R-related to $b$ (write $aRb$)
    • $(a, b) \notin R$ → $a$ is not related to $b$
  • If $R$ is a relation from $A$ to itself → $R$ is called a relation on $A$

Examples of Relations

  • Example 1: $A = \{1, 3, 5, 7\}$, $B = \{6, 8\}$, $R$ = "less than" from $A$ to $B$
    • $R = \{(1, 6), (1, 8), (3, 6), (3, 8), (5, 6), (5, 8), (7, 8)\}$
  • Example 2: $A = \{1, 2, 3, 4, 5\}$, $B = \{a, b, c\}$
    • Relations from $A$ into $B$:
      • $R_1 = \{(1, a)\}$
      • $R_2 = \{(2, b), (3, b), (4, c), (5, a)\}$
      • $R_3 = \{(1, a), (2, b), (3, c)\}$
    • Relations from $B$ to $A$:
      • $R_4 = \{(a, 3), (b, 1)\}$
      • $R_5 = \{(b, 2), (c, 4), (a, 2), (b, 3)\}$
      • $R_6 = \{(b, 5)\}$
Definition: Let $R$ be a relation from $A$ into $B$. Then,
  • The domain of $R$, denoted by $\text{Dom}(R)$, is the set of first coordinates of the elements of $R$: $$\text{Dom}(R) = \{a \in A : (a, b) \in R\}$$
  • The range of $R$, denoted by $\text{Range}(R)$, is the set of second coordinates of elements of $R$: $$\text{Range}(R) = \{b \in B : (a, b) \in R\}$$
Remark: If $R$ is a relation from $A$ to $B$, then $B$ is called the codomain of $R$. The range is always a subset of the codomain.

Example 1

Question: Given $R = \{(4,7), (5,8), (6,10)\}$ is a relation from $A = \{1,2,3,4,5,6\}$ to $B = \{6,7,8,9,10\}$. Find the domain, range, and codomain of $R$.
  • Solution:
    • $\text{Dom}(R) = \{4,5,6\}$
    • $\text{Range}(R) = \{7,8,10\}$
    • $\text{Codomain}(R) = \{6,7,8,9,10\}$
Question: Given $R = \{(8,2), (6,-3), (5,7), (5,-3)\}$ is a relation between $\{5,6,8\}$ and $\{2,-3,7\}$. Find the domain and range of $R$.
  • Solution:
    • $\text{Dom}(R) = \{5,6,8\}$
    • $\text{Range}(R) = \{2,-3,7\}$
Important Remarks:
  • If $(a,b) \in R$, then $b$ is called the image of $a$, while $a$ is called the pre-image of $b$
  • If domain and/or range is infinite, use set builder notation
  • Example: $R = \{(x, y) : y = 2x - 3, x, y \in \mathbb{R}\}$
    • $(0,-3)$, $(0.5,-2)$, $(-2,-7)$ are examples of ordered pairs in the relation

Example 2

Question (a): Let $A = \{1,2,3,4,6\}$ and $R$ be the relation on $A$ defined by $R = \{(a, b) : a,b \in A, a \text{ is a factor of } b\}$. Find $\text{Dom}(R)$ and $\text{Range}(R)$.
  • Solution:
    • $R = \{(1,1), (1,2), (1,3), (1,4), (1,6), (2,2), (2,4), (2,6), (3,3), (3,6), (4,4), (6,6)\}$
    • $\text{Dom}(R) = \{1,2,3,4,6\}$
    • $\text{Range}(R) = \{1,2,3,4,6\}$
Question (b): Let $A = \{1,2,3,4,5\}$, $B = \{1,2,3,\dots,67\}$, and $R = \{(x,y) \in A \times B : x \text{ is cube root of } y\}$. Find $R$, $\text{Dom}(R)$, and $\text{Range}(R)$.
  • Solution:
    • $1 = \sqrt[3]{1}$, $2 = \sqrt[3]{8}$, $3 = \sqrt[3]{27}$, $4 = \sqrt[3]{64}$ (all in $B$)
    • $5 = \sqrt[3]{125}$ (not in $B$)
    • $R = \{(1,1), (2,8), (3,27), (4,64)\}$
    • $\text{Dom}(R) = \{1,2,3,4\}$
    • $\text{Range}(R) = \{1,8,27,64\}$

Special Relations

  • A relation $R$ on a set $A$ is called:
    • Universal relation → $R = A \times A$
    • Identity relation → $R = \{(a,a) : a \in A\}$
    • Void or empty relation → $R = \emptyset$

Inverse of a Relation

Definition: If $R$ is a relation from $A$ to $B$, then the inverse relation of $R$, denoted by $R^{-1}$, is a relation from $B$ to $A$ defined as: $$R^{-1} = \{(y,x) : (x,y) \in R\}$$
  • $\text{Dom}(R) = \text{Range}(R^{-1})$
  • $\text{Range}(R) = \text{Dom}(R^{-1})$
  • Example: If $R = \{(1,4), (9,15), (10,2)\}$ is a relation on $A = \{1,2,3,\dots,20\}$
    • $R^{-1} = \{(4,1), (15,9), (2,10)\}$

Example 3

Question: Let $R$ be a relation defined on $\mathbb{N}$ by $R = \{(a,b) : a,b \in \mathbb{N}, a + 2b = 11\}$. Find:
  • $R$
  • $\text{Dom}(R)$
  • $\text{Range}(R)$
  • $R^{-1}$
  • Solution:
    • Find all ordered pairs:
      • $b=1 \rightarrow a=9$
      • $b=2 \rightarrow a=7$
      • $b=3 \rightarrow a=5$
      • $b=4 \rightarrow a=3$
      • $b=5 \rightarrow a=1$
      • $b=6 \rightarrow a=-1 \notin \mathbb{N}$
    • $R = \{(9,1), (7,2), (5,3), (3,4), (1,5)\}$
    • $\text{Dom}(R) = \{1,3,5,7,9\}$
    • $\text{Range}(R) = \{1,2,3,4,5\}$
    • $R^{-1} = \{(1,9), (2,7), (3,5), (4,3), (5,1)\}$

Functions

  • Mathematically, it is important to distinguish among relations that assign a unique range element to each domain element and those that do not
Definition: A function is a relation in which each element of the domain corresponds to exactly one element of the range.
Example: Determine whether the following relations are functions.
  • a) $R = \{(5, -2), (3, 5), (3, 7)\}$
  • b) $\{(2, 4), (3, 4), (6, -4)\}$
  • Solution:
    • a) Since the domain element $3$ is assigned to two different values in the range, $5$ and $7$, it is not a function
    • b) Each element in the domain $\{2, 3, 6\}$ is assigned no more than one value → $2$ is assigned only $4$, $3$ is assigned only $4$, and $6$ is assigned only $-4$. Therefore, it is a function
Remark: Map, mapping, transformation, and correspondence are synonyms for the word function. If $f$ is a function and $(x, y) \in f$, we say $x$ is mapped to $y$.
Definition: A relation $f$ from $A$ into $B$ is called a function from $A$ into $B$, denoted by $f: A \rightarrow B$ or $A \xrightarrow{f} B$, if and only if:
  • (i) $\text{Dom}(f) = A$
  • (ii) No element of $A$ is mapped by $f$ to more than one element in $B$, i.e. if $(x, y) \in f$ and $(x, z) \in f$, then $y = z$
Important Remarks:
  • If $x \in A$ corresponds to $y \in B$ under function $f$, then $f(x) = y$
  • $y$ is called the image of $x$ under $f$
  • $x$ is called a pre-image of $y$ under $f$
  • The symbol $f(x)$ is read as "$f$ of $x$" but not "$f$ times $x$"
  • To show a relation $f$ from $A$ into $B$ is a function:
    • First show $\text{Dom}(f) = A$
    • Next show $f$ is well defined or single-valued, i.e. if $x = y$ in $A$, then $f(x) = f(y)$ in $B$ for all $x, y \in A$
Example: Let $A = \{1, 2, 3, 4\}$ and $B = \{1, 6, 8, 11, 15\}$. Which of the following are functions from $A$ to $B$?
  • a) $f$ defined by $f(1) = 1$, $f(2) = 6$, $f(3) = 8$, $f(4) = 8$
  • b) $f$ defined by $f(1) = 1$, $f(2) = 6$, $f(3) = 15$
  • c) $f$ defined by $f(1) = 6$, $f(2) = 6$, $f(3) = 6$, $f(4) = 6$
  • d) $f$ defined by $f(1) = 1$, $f(2) = 6$, $f(3) = 8$, $f(4) = 11$
  • e) $f$ defined by $f(1) = 1$, $f(2) = 8$, $f(3) = 11$, $f(4) = 15$
  • Solution:
    • a) Function → to each element of $A$ corresponds exactly one element of $B$
    • b) Not a function → no element of $B$ corresponds to $4 \in A$
    • c) Function → to each element of $A$ corresponds exactly one element of $B$ (images of all elements are the same)
    • d) Not a function → two elements of $B$ correspond to $2$ (image of $2$ is not unique)
    • e) Function → to each element of $A$ corresponds exactly one element of $B$
Key Point: As with relations, we can describe a function with an equation. For example, $y = 2x + 1$ is a function, since each $x$ will produce only one $y$.
  • Example: Let $f = \{(x, y) : y = x^2\}$
    • $1 \rightarrow 1$, $-1 \rightarrow 1$
    • $2 \rightarrow 4$, $-2 \rightarrow 4$
    • $3 \rightarrow 9$, $-3 \rightarrow 9$
    • Any real number $x$ is mapped to its square
    • As the square of a number is unique, $f$ maps every real number to a unique number → $f$ is a function from $\mathbb{R}$ into $\mathbb{R}$
Vocabulary:
  • Independent variable → variable representing possible values in the domain ($x$)
  • Dependent variable → variable representing possible values in the range ($y$)
  • In ordered pair notation $(x, y)$: $x$ is the independent variable and $y$ is the dependent variable

Domain, Codomain, and Range of a Function

Definition: For the function $f: A \rightarrow B$:
  • (i) The set $A$ is called the domain of $f$
  • (ii) The set $B$ is called the codomain of $f$
  • (iii) The set $\{f(x) : x \in A\}$ of all images of elements of $A$ is called the range of $f$
Example: Let $A = \{1, 2, 3\}$ and $B = \{1, 2, 3, \dots, 10\}$. Let $f: A \rightarrow B$ be the correspondence which assigns to each element in $A$, its square.
  • $f(1) = 1$, $f(2) = 4$, $f(3) = 9$
  • $\text{Dom}(f) = \{1, 2, 3\}$
  • $\text{Range}(f) = \{1, 4, 9\}$
  • $\text{Codomain}(f) = \{1, 2, 3, \dots, 10\}$
Example: Let $A = \{2, 4, 6, 7, 9\}$, $B = \mathbb{N}$. Let $f: A \rightarrow B$ be a function defined by $f(x) = 15x + 17$, $x \in A$.
  • $f(2) = 15(2) + 17 = 47$
  • $f(4) = 15(4) + 17 = 77$
  • $f(6) = 15(6) + 17 = 107$
  • $f(7) = 15(7) + 17 = 122$
  • $f(9) = 15(9) + 17 = 152$
  • $\text{Dom}(f) = \{2, 4, 6, 7, 9\}$
  • $\text{Range}(f) = \{47, 77, 107, 122, 152\}$
  • $\text{Codomain}(f) = \mathbb{N}$
Example: Let $f$ be the subset of $\mathbb{Q} \times \mathbb{Z}$ defined by $f = \left\{\left(\frac{p}{q}, p\right) : p, q \in \mathbb{Z}, q \neq 0\right\}$. Is $f$ a function?
  • Solution:
    • $\text{Dom}(f) = \mathbb{Q}$ → condition (i) is satisfied
    • $\left(\frac{2}{3}, 2\right) \in f$, $\left(\frac{4}{6}, 4\right) \in f$ and $\frac{2}{3} = \frac{4}{6}$ but $f\left(\frac{2}{3}\right) = 2 \neq 4 = f\left(\frac{4}{6}\right)$
    • Thus $f$ is not well defined
    • Hence, $f$ is not a function from $\mathbb{Q}$ to $\mathbb{Z}$
Example: Let $f$ be the subset of $\mathbb{Z} \times \mathbb{Z}$ defined by $f = \{(mn, m + n) : m, n \in \mathbb{Z}\}$. Is $f$ a function?
  • Solution:
    • Let $x \in \mathbb{Z}$. Then $x = x \cdot 1$ → $(x, x+1) = (x \cdot 1, x+1) \in f$
    • Thus $\mathbb{Z} \subseteq \text{Dom}(f)$ and $\text{Dom}(f) \subseteq \mathbb{Z}$ → $\text{Dom}(f) = \mathbb{Z}$
    • Now, $4 \in \mathbb{Z}$ and $4 = 4 \cdot 1 = 2 \cdot 2$
    • $(4 \cdot 1, 4+1)$ and $(2 \cdot 2, 2+2)$ are in $f$
    • $4 \cdot 1 = 2 \cdot 2$ but $f(4 \cdot 1) = 5 \neq 4 = f(2 \cdot 2)$
    • Thus $f$ is not well defined
    • Hence, $f$ is not a function from $\mathbb{Z}$ to $\mathbb{Z}$
Example: Determine whether the following equations determine $y$ as a function of $x$. If so, find the domain.
  • a) $y = -3x + 5$
  • b) $y = \frac{2x}{3x - 5}$
  • c) $y^2 = x$
  • Solution:
    • a) $y = -3x + 5$: For each $x$ there is a unique $y$ → function
    • b) $y = \frac{2x}{3x - 5}$: For each $x$ there is a unique $y$ → function
      • Domain: $3x - 5 \neq 0 \rightarrow x \neq \frac{5}{3}$
      • $\text{Dom}(f) = \{x : x \neq \frac{5}{3}\}$
    • c) $y^2 = x$: If $x = 9$, then $y^2 = 9 \rightarrow y = \pm 3$ → two $y$-values
    • Thus $y^2 = x$ is not a function
Example: Find the domain of the function $y = \sqrt{3x - x^2}$.
  • Solution:
    • For $y$ to be real, the expression under the radical must be non-negative
    • $3x - x^2 \geq 0 \rightarrow x(3 - x) \geq 0$
    • Sign analysis → $x(3 - x) \geq 0$ when $0 \leq x \leq 3$
    • $\text{Dom}(f) = \{x : 0 \leq x \leq 3\}$ or $[0, 3]$

Real Valued Functions and Their Properties
Understanding Real Functions and Operations on Functions
  • Let $f$ be a function from set $A$ to set $B$
  • If $B \subseteq \mathbb{R}$, then $f$ is called a real valued function
  • If $A \subseteq \mathbb{R}$ and $B \subseteq \mathbb{R}$, then $f: A \rightarrow B$ is called a real function
Example:
  • $f: \mathbb{R} \rightarrow \mathbb{R}$ defined by $f(x) = x^2 + 3x + 7$, $x \in \mathbb{R}$ is a real function
  • $f: \mathbb{R} \rightarrow \mathbb{R}$ defined as $f(x) = |x|$ is a real valued function

Operations on Functions

  • Functions are not numbers, but they can be added, subtracted, multiplied, and divided
  • Example: $f(x) = \frac{x-3}{2}$, $g(x) = \sqrt{x}$
    • $(f + g)(x) = f(x) + g(x) = \frac{x-3}{2} + \sqrt{x}$
Definition: Let $f(x)$ and $g(x)$ be two functions. We define:
  • 1. $(f + g)(x) = f(x) + g(x)$ → Sum
  • 2. $(f - g)(x) = f(x) - g(x)$ → Difference
  • 3. $(f \cdot g)(x) = f(x)g(x)$ → Product
  • 4. $(f/g)(x) = \frac{f(x)}{g(x)}$ → Quotient (provided $g(x) \neq 0$)
Domain Rule: For sum, difference, and product:
  • $\text{Dom}(f + g) = \text{Dom}(f) \cap \text{Dom}(g)$
  • Same for $(f - g)$ and $(f \cdot g)$
  • For quotient: $\text{Dom}(f/g) = [\text{Dom}(f) \cap \text{Dom}(g)] \setminus \{x : g(x) = 0\}$
Example: Let $f(x) = 3x^2 + 2$ and $g(x) = 5x - 4$. Find each of the following and its domain.
  • a) $(f + g)(x)$
  • b) $(f - g)(x)$
  • c) $(f \cdot g)(x)$
  • d) $(f/g)(x)$
  • Solution:
    • a) $(f + g)(x) = (3x^2 + 2) + (5x - 4) = 3x^2 + 5x - 2$
    • b) $(f - g)(x) = (3x^2 + 2) - (5x - 4) = 3x^2 - 5x + 6$
    • c) $(f \cdot g)(x) = (3x^2 + 2)(5x - 4) = 15x^3 - 12x^2 + 10x - 8$
    • d) $(f/g)(x) = \frac{3x^2 + 2}{5x - 4}$
    • $\text{Dom}(f + g) = \text{Dom}(f - g) = \text{Dom}(fg) = \mathbb{R} \cap \mathbb{R} = \mathbb{R}$
    • $\text{Dom}(f/g) = \mathbb{R} \setminus \{\frac{4}{5}\}$
Example: Let $f(x) = \sqrt[4]{x + 1}$ and $g(x) = \sqrt{9 - x^2}$, with respective domains $[-1, \infty)$ and $[-3, 3]$. Find formulas for $f + g$, $f - g$, $f \cdot g$, $f/g$, and $f^3$, and give their domains.
Formula Domain
$(f + g)(x) = \sqrt[4]{x+1} + \sqrt{9 - x^2}$ $[-1, 3]$
$(f - g)(x) = \sqrt[4]{x+1} - \sqrt{9 - x^2}$ $[-1, 3]$
$(f \cdot g)(x) = \sqrt[4]{x+1} \cdot \sqrt{9 - x^2}$ $[-1, 3]$
$(f/g)(x) = \frac{\sqrt[4]{x+1}}{\sqrt{9 - x^2}}$ $[-1, 3)$
$f^3(x) = (f(x))^3 = (\sqrt[4]{x+1})^3 = (x+1)^{3/4}$ $[-1, \infty)$
Note: There is yet another way of producing a new function from two given functions.

Composition of Functions

Definition: Given two functions $f(x)$ and $g(x)$, the composition of the two functions is denoted by $f \circ g$ and is defined by: $$(f \circ g)(x) = f(g(x))$$
  • $(f \circ g)(x)$ is read as "$f$ composed with $g$ of $x$"
  • The domain of $f \circ g$ consists of those $x$'s in the domain of $g$ whose range values are in the domain of $f$
Example: Suppose $f = \{(2, z), (3, q)\}$ and $g = \{(a, 2), (b, 3), (c, 5)\}$.
  • $(f \circ g)(a) = f(g(a)) = f(2) = z$
  • $(f \circ g)(b) = f(g(b)) = f(3) = q$
  • $(f \circ g)(c) = f(g(c)) = f(5)$ → $5 \notin \text{Dom}(f)$ so undefined
  • Thus $f \circ g = \{(a, z), (b, q)\}$
Example: Given $f(x) = 5x^2 - 3x + 2$ and $g(x) = 4x + 3$, find:
  • a) $(f \circ g)(-2)$
  • b) $(g \circ f)(2)$
  • c) $(f \circ g)(x)$
  • d) $(g \circ f)(x)$
  • Solution:
    • a) $(f \circ g)(-2) = f(g(-2))$
      • $g(-2) = 4(-2) + 3 = -5$
      • $f(-5) = 5(-5)^2 - 3(-5) + 2 = 142$
    • b) $(g \circ f)(2) = g(f(2))$
      • $f(2) = 5(2)^2 - 3(2) + 2 = 16$
      • $g(16) = 4(16) + 3 = 67$
    • c) $(f \circ g)(x) = f(g(x)) = f(4x + 3)$
      • $= 5(4x + 3)^2 - 3(4x + 3) + 2$
      • $= 80x^2 + 108x + 38$
    • d) $(g \circ f)(x) = g(f(x)) = g(5x^2 - 3x + 2)$
      • $= 4(5x^2 - 3x + 2) + 3$
      • $= 20x^2 - 12x + 11$
Example: Given $f(x) = \frac{x}{x+1}$ and $g(x) = \frac{2}{x-1}$, find:
  • a) $(f \circ g)(x)$ and its domain
  • b) $(g \circ f)(x)$ and its domain
  • Solution:
    • a) $(f \circ g)(x) = f\left(\frac{2}{x-1}\right) = \frac{\frac{2}{x-1}}{\frac{2}{x-1} + 1} = \frac{2}{x+1}$
      • $\text{Dom}(f \circ g) = \{x : x \neq \pm 1\}$
    • b) $(g \circ f)(x) = g\left(\frac{x}{x+1}\right) = \frac{2}{\frac{x}{x+1} - 1} = -2x - 2$
      • $\text{Dom}(g \circ f) = \{x : x \neq -1\}$
Example: Let $f(x) = \frac{6x}{x^2 - 9}$ and $g(x) = \sqrt{3x}$. Find $(f \circ g)(12)$ and $(g \circ f)(x)$ and its domain.
  • Solution:
    • $(f \circ g)(12) = f(g(12)) = f(\sqrt{36}) = f(6) = \frac{36}{36 - 9} = \frac{4}{3}$
    • $(f \circ g)(x) = f(\sqrt{3x}) = \frac{6\sqrt{3x}}{(\sqrt{3x})^2 - 9} = \frac{6\sqrt{3x}}{3x - 9} = \frac{2\sqrt{3x}}{x - 3}$
    • $\text{Dom}(f \circ g) = [0, 3) \cup (3, \infty)$

Equality of Functions

  • Let $f: A \rightarrow B$ and $g: A \rightarrow B$ be two functions
  • $f$ and $g$ are subsets of $A \times B$
  • Suppose $f = g$. Let $x \in A$. Then $(x, f(x)) \in f = g$ and thus $(x, f(x)) \in g$
  • Since $g$ is a function and $(x, f(x)), (x, g(x)) \in g$, we must have $f(x) = g(x)$
  • Conversely, assume $g(x) = f(x)$ for all $x \in A$
  • Let $(x, y) \in f$. Then $y = f(x) = g(x)$. Thus $(x, y) \in g$, which implies $f \subseteq g$
  • Similarly, $g \subseteq f$. It follows that $f = g$
Definition: Two functions are said to be equal if and only if the following two conditions hold:
  • i) The functions have the same domain
  • ii) Their functional values are equal at each element of the domain
Example: Let $f: \mathbb{Z} \rightarrow \mathbb{Z}^+ \cup \{0\}$ and $g: \mathbb{Z} \rightarrow \mathbb{Z}^+ \cup \{0\}$ be defined by $f = \{(n, n^2) : n \in \mathbb{Z}\}$ and $g = \{(n, |n|^2) : n \in \mathbb{Z}\}$.
  • For all $n \in \mathbb{Z}$, $f(n) = n^2 = |n|^2 = g(n)$
  • Thus, $f = g$
Example: Let $f(x) = \frac{x^2 - 25}{x - 5}$, $x \in \mathbb{R} \setminus \{5\}$, and $g(x) = x + 5$, $x \in \mathbb{R}$.
  • $\text{Dom}(f) = \mathbb{R} \setminus \{5\}$
  • $\text{Dom}(g) = \mathbb{R}$
  • Since $\text{Dom}(f) \neq \text{Dom}(g)$, the functions are not equal
Key Point: Two functions $f: A \rightarrow B$ and $g: A \rightarrow B$ are equal if and only if $f(x) = g(x)$ for all $x \in A$. Both the domain and the functional values must match.

Types of Functions and Inverse of a Function
One-to-One, Onto, and One-to-One Correspondence

One-to-One Functions

Definition: A function $f: A \rightarrow B$ is called one-to-one (often written $1-1$) if and only if for all $x_1, x_2 \in A$: $$f(x_1) = f(x_2) \implies x_1 = x_2$$
  • In words: No two elements of $A$ are mapped to one element of $B$
Example: Let $A = \{1, 2, 3, 4, 5, 6\}$ and $B = \{7, a, b, c, d, 8, e\}$.
  • $f = \{(1, 7), (2, a), (3, b), (4, b), (5, c), (6, 8)\}$ → not 1-1 because $f(3) = f(4)$ but $3 \neq 4$
  • $g = \{(1, 7), (2, a), (3, b), (4, c), (5, 8), (6, d)\}$ → 1-1 function
Example: Let $A = \{1, 2, 3, 4\}$ and $B = \{1, 4, 7, 8\}$.
  • $f: A \rightarrow B$ defined as $f(1) = 1$, $f(2) = 4$, $f(3) = 4$, $f(4) = 8$ → not 1-1
  • $g: A \rightarrow B$ defined as $g(1) = 4$, $g(2) = 7$, $g(3) = 1$, $g(4) = 8$ → 1-1 function

Onto Functions

Definition: Let $f$ be a function from a set $A$ into a set $B$. Then $f$ is called an onto function (or $f$ maps onto $B$) if every element of $B$ is image of some element in $A$: $$\text{Range}(f) = B$$
Example: Let $A = \{1, 2, 3\}$ and $B = \{1, 4, 5\}$.
  • $f: A \rightarrow B$ defined as $f(1) = 1$, $f(2) = 5$, $f(3) = 1$ → not onto because no element maps to $4$
  • $g: A \rightarrow B$ given by $g = \{(1, 4), (2, 5), (3, 1)\}$ → onto because each element of $B$ is an image
  • If $A$ is a non-empty set, the function $i_A: A \rightarrow A$ defined by $i_A(x) = x$ for all $x \in A$ is a $1-1$ function from $A$ onto $A$
  • $i_A$ is called the identity map on $A$
Example: Consider $f: \mathbb{Z} \rightarrow \mathbb{Z}$ defined by $f(n) = n^2$ for all $n \in \mathbb{Z}$.
  • $f$ is a function (well-defined)
  • $f(1) = 1 = f(-1)$ and $1 \neq -1$ → not 1-1
  • For all $n \in \mathbb{Z}$, $f(n) \geq 0$ → negative integers have no preimage → not onto
  • Note: $f$ is onto $\{0, 1, 4, 9, \dots\}$
Example: Consider $f: \mathbb{Z} \rightarrow \mathbb{Z}$ defined by $f(n) = 2n$ for all $n \in \mathbb{Z}$.
  • $f$ is a function
  • If $f(n) = f(n')$, then $2n = 2n' \rightarrow n = n'$ → 1-1
  • For all $n \in \mathbb{Z}$, $f(n)$ is even → odd integers have no preimage → not onto

One-to-One Correspondence

Definition: A function $f: A \rightarrow B$ is said to be a 1-1 correspondence if $f$ is both 1-1 and onto.
Example: Let $A = \{0, 1, 2, 3, 4, 5\}$ and $B = \{0, 5, 10, 15, 20, 25\}$. Suppose $f: A \rightarrow B$ given by $f(x) = 5x$ for all $x \in A$.
  • Every element of $B$ has a preimage in $A$ → onto
  • If $f(x) = f(y)$, then $5x = 5y \rightarrow x = y$ → 1-1
  • Therefore, $f$ is a 1-1 correspondence between $A$ and $B$
Example: Let $A$ be a finite set. If $f: A \rightarrow A$ is onto, then it is one-to-one.
  • Solution:
    • Let $A = \{a_1, a_2, \dots, a_n\}$
    • $\text{Range}(f) = \{f(a_1), f(a_2), \dots, f(a_n)\}$
    • Since $f$ is onto, $\text{Range}(f) = A$
    • $A = \{f(a_1), f(a_2), \dots, f(a_n)\}$ → the values are all distinct
    • Hence, $a_i \neq a_j \implies f(a_i) \neq f(a_j)$ → $f$ is $1-1$

Inverse of a Function

  • Since a function is a relation, the inverse of a function $f$ is denoted by $f^{-1}$ and is defined by:
    • $f^{-1} = \{(y, x) : (x, y) \in f\}$
  • Example: If $f = \{(2,4), (3,6), (1,7)\}$, then $f^{-1} = \{(4,2), (6,3), (7,1)\}$
  • Note: The inverse of a function is not always a function
    • Example: $f = \{(2,4), (3,6), (5,4)\}$ → $f^{-1} = \{(4,2), (6,3), (4,5)\}$ which is not a function
  • Only one-to-one functions have inverses that are also functions
Key Point: A one-to-one function is special because only one-to-one functions have inverse functions. If a function is not one-to-one, it does not have an inverse function.

Steps to Find the Inverse of a Function

  • If a function is one-to-one, to find the inverse:
    • 1. Interchange $x$ and $y$ in the equation $y = f(x)$
    • 2. Solve the resulting equation for $y$
  • Domain and Range:
    • $\text{Dom}(f^{-1}) = \text{Range}(f)$
    • $\text{Range}(f^{-1}) = \text{Dom}(f)$
Example: Given $y = f(x) = x^3$. Find $f^{-1}$ and its domain.
  • Solution:
    • $y = x^3$ → Interchange $x$ and $y$: $x = y^3$
    • Take cube root of both sides: $\sqrt[3]{x} = y$
    • Thus, $f^{-1}(x) = \sqrt[3]{x}$
    • $\text{Dom}(f^{-1}) = \mathbb{R}$
Example: Let $y = f(x) = \frac{x}{x+2}$. Find $f^{-1}(x)$.
  • Solution:
    • $y = \frac{x}{x+2}$ → Interchange $x$ and $y$: $x = \frac{y}{y+2}$
    • Solve for $y$: $x(y+2) = y \rightarrow xy + 2x = y$
    • $2x = y - xy \rightarrow 2x = y(1-x)$
    • $y = \frac{2x}{1-x}$
    • Thus, $f^{-1}(x) = \frac{2x}{1-x}$
Important Remark:
  • $f^{-1}(x)$ is not the reciprocal of $f(x)$
  • $f^{-1}(x) \neq \frac{1}{f(x)}$
  • To write the reciprocal using a negative exponent: $\frac{1}{f(x)} = [f(x)]^{-1}$
